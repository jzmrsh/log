# 📝 JZMRSH LOG

O repositório central que alimenta o meu **micro-blog/registro de atividades**.

Acesse a versão live em: **[jzmrsh.com/log](https://jzmrsh.com/log)**

---

### 🚀 O Conceito

Este log funciona como um **fluxo de consciência técnico** e um registro cronológico de aprendizados, notas de projetos e pensamentos curtos. É um espaço onde a utilidade e a simplicidade vêm antes do design complexo.

### 🛠️ Estrutura do Projeto

O site foca em uma arquitetura limpa:

* **Minimalismo:** Sem trackers, sem anúncios, sem excesso de JavaScript.
* **Cronológico:** Entradas listadas por data, facilitando o acompanhamento da evolução de ideias.
* **Markdown-first:** Conteúdo escrito de forma simples e direta.

### 📂 Como Funciona

As entradas são organizadas de forma que cada arquivo represente um momento ou uma descoberta técnica.

> "Keep it simple. Write it down. Move forward."
